- [x] Configuração inicial do repositório (Backend & Frontend)
- [x] Dockerização do ambiente (Django, React, Postgres, Redis)
- [x] Definição de Arquitetura e Guia de Desenvolvimento

## 🏠 2. Lobby & Gestão de Partida
- [ ] Criação de Salas (UUID)
- [ ] Entrada de Jogadores via Mobile (PWA)
- [ ] Definição de Líder da Sala e Início de Jogo

- [ ] Interface de criação de plano (Entrada, Ferramentas, Rota, Fuga)

- [ ] Integração com IA para validação e geração de sementes de provas

- [ ] Seleção automática do plano vencedor (Crime)
- [ ] Animação/Display do incidente na TV
- [ ] Transição de estado para Investigação

- [ ] Sistema de Pontos de Investigação (PI) reativo
- [ ] Lógica de Líder da Rodada (12 PI/rodada)
- [ ] Ações de Investigação (Perícia, Logs, Câmeras, etc.)
- [ ] Sincronização em tempo real: Ação no Mobile -> Evidência no Mural da TV

## 🎭 6. Poderes do Culpado
- [ ] Interface privada para o Culpado
- [ ] Ações de Sabotagem (Pistas ambíguas)
- [ ] Geração de Falsas Informações via IA

## ⚖️ 7. Fase de Solução & IA
- [ ] Interface de Palpite Final (Múltiplos campos)
- [ ] Lógica de Avaliação da IA (Cálculo de Score %)
- [ ] Telas de Vitória/Derrota e Resumo da Partida

## 🎨 8. Polimento & UX
- [ ] Design System "Premium" (Dark Mode, Neon, Glassmorphism)
- [ ] Micro-animações de transição de turno
- [ ] Efeitos sonoros e feedback tátil no mobile
